PROJECT OVERVIEW

The Interactive Exhibition Map Dashboard is a web-based map that helps visitors find buildings, exhibits, and services inside an exhibition. It works on kiosks, mobiles, and desktops. Users can search, explore the map, and get navigation easily.

MAIN OBJECTIVES

Help visitors find locations quickly
Provide easy navigation with GPS
Allow searching for exhibits and facilities
Work smoothly on all device types
Reduce the need for staff to give directions
Give real-time updates and information

KEY FEATURES
Fast Search: Suggestions, filters, and offline support
Interactive Map: Zoom, pan, clickable buildings, colored markers
Navigation: Real-time location, routes, walking time
Responsive UI: Works on kiosk, tablet, and mobile
Content System: Building details, images, schedules, events

TECHNOLOGY USED
Frontend: React, Leaflet.js, SVG maps, Socket.io
Backend: REST API + WebSocket
PWA: Works offline and can be installed
Data: JSON mapping, building database, search indexing

USER EXPERIENCE
Easy to use for first-time visitors
Clean design with simple controls
Touch-friendly gestures
Clear information display with smooth animations

DEPLOYMENT MODES
Kiosk: Full-screen, big buttons, auto-reset
Mobile: GPS navigation, save locations, offline mode
Staff: Dashboard for updates and analytics

BENEFITS
Visitors find places faster
Staff workload reduces
Better space planning through analytics
Lower cost for printed maps
Improved visitor satisfaction
